图例说明
蓝色方框 进程1: vla_bridge_node — Python 3.10 + ROS2 Humble（原设计范围）
绿色方框 进程2: vla_inference_server — Python 3.12 + lerobot3（新增，原设计内嵌于进程1）
橙色圆形 multiprocessing.Pipe IPC — 新增，替代原设计的直接函数调用
组件原设计新方案
SmolVlaPolicy 在 vla_bridge_node 进程内，直接 import lerobot 移至独立 Python 3.12 进程
PREPROC / POSTPROC 在 VLA_Inference 线程（Python 3.10） 移至进程2（Python 3.12）
推理调用方式 线程内同步函数调用 Pipe.send(obs) → Pipe.recv(action)（跨进程）
VLA_Inference 线程 独立推理线程（与 ROS 线程共进程） 改为 InferenceClient（负责 Pipe 通信）
ROS Topic 接口 完全不变，外部节点无感知
